# Menu 菜单

::: tip 规划中
Menu 通过自动代理机制透传 `el-menu`，暂无自定义扩展。如需查看完整 API，请参考 [Element Plus Menu 文档](https://element-plus.org/zh-CN/component/menu.html)。
:::

## 基础用法

```vue
<zq-menu default-active="1" mode="horizontal">
  <zq-menu-item index="1">导航一</zq-menu-item>
  <zq-menu-item index="2">导航二</zq-menu-item>
</zq-menu>
```
